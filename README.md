# Work-Schedule

Telepítés
1. Klónozd a repót a gépedre:
   ```bash
   git clone https://github.com/balazsNorbert/Work-Schedule.git
2. Navigálj a mappába és nyisd meg VSCode-ban:
  cd Work-Schedule
3. Futtasd npm run dev-el terminálba

Használat
1.Kiválaszthatod a jobb felső sarokba hogy kihez rendeled a feladatod.
2.Válaszd ki a hónapot, amelyhez feladatokat szeretnél hozzáadni.
3.Írd be a célokat a megfelelő mezőbe, majd nyomd meg az Enter billentyűt a mentéshez.
4.A hozzáadott személyhez fűződő feladatok megjelennek a hónap alatt.
5.Törölheted a feladatokat a szemetes gombra kattintva.
6 A hónap nevére kattintva megjelennek a hónap napjai amire szintén lehet kattintani nap szerinti feladatok céljából, de ez még nem működő képes.

Technológiák
1.HTML: Az alkalmazás alapstruktúrája.
2.Tailwind CSS: Az alkalmazás stílusának meghatározása.
3.JavaScript: Az interaktivitás biztosítása és a feladatok kezelése.
4.jQuery: A DOM manipuláció egyszerűsítése.
5.Material-UI: Beépitett ikon használat.
6.Adatok tárolása: LocalStorage-ben.

Fejlesztés
1.A napok szerinti célok elmentése személyre szabottan.
2.Ha egy napra kattintunk lehessen részletesebb információkat elmenteni órákra beosztva.
3.Ezek csoportositani valami kategoria szerint pld. fonossági sorrend.
3.UI fejlesztése

Időmenedzsment
7,5-8 óra között. Probáltam minnél jobban kihasználni az időmet de most ennyit sikerült megcsinálni.
